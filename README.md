# ✨ About Learnexnus AI

Learnexnus AI is a next-generation AI-powered academic ecosystem designed to help students learn smarter, practice better, and grow faster.

The platform combines intelligent note generation, AI-powered question creation, visual learning roadmaps, and smart document analysis into one unified experience.

---

# 🚀 Features

<div id="features"></div>

| Feature | Description |
|----------|-------------|
| 🧠 AI Notes Generator | Generate smart structured notes instantly using Gemini AI |
| ❓ AI Question Generator | Create MCQs, short answers, long answers, and conceptual questions |
| 📄 PDF & Image Analysis | Upload PDFs, screenshots, handwritten notes, or question papers |
| 🔍 OCR Text Extraction | Extract readable text using AI-powered OCR |
| 🛣 Career Roadmaps | Visual career guidance and learning paths |
| 🌐 Interactive Flowcharts | React Flow + Mermaid.js based visual learning systems |
| 📚 Smart History System | Save and reopen generated notes, questions, and roadmaps |
| 📥 PDF Export | Export generated content professionally |
| ⚡ Gemini AI Integration | AI-powered academic intelligence engine |
